# linajia-excel-tool

获取链家指定城市、指定地区楼盘信息，并导出 excel 表格方便查看。

## 使用

安装依赖

```js
pnpm install
```

执行脚本文件

```js
pnpm start
```

脚本执行完毕会将 `excel` 表格输出至 `dist` 目录，图片会输出至 `imgs` 目录。
